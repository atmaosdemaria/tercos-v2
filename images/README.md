## Sobre o Projeto

Este é o portfólio online do **Ateliê Mãos de Maria**, um espaço artesanal dedicado à confecção de terços únicos, feitos com amor e devoção. Cada peça é tecida manualmente, conta por conta, como uma oração em forma de arte.

### O que você encontra aqui:
- Galeria de terços artesanais com fotos em alta qualidade  
- Zoom nas imagens para ver detalhes das contas e crucifixos  
- Preços e descrições de cada modelo  
- Botão direto para comprar ou conversar via WhatsApp  

## Tecnologias Utilizadas
- HTML5 + CSS3 (puro, sem frameworks pesados)  
- JavaScript vanilla (para o modal de zoom nas fotos)  
- Design responsivo (funciona perfeitamente em celular, tablet e desktop)
